# CheckerMouse

A simple checkers interface built with Unity. It uses a bitboard-based backend for move validation and has an embedded checkers engine (written in c++) to play against the computer.

Download from [itch](https://m-8000.itch.io/checkermouse)

## Features

* **Play against computer:** A checkers engine that can evaluate to a depth of ~16 moves.
* **Timed Play:** Fully functional clock with support for increments.
* **Move History:** Saves the history of the game so you can export it.

## How to Use

### Playing a Game

By default, the game starts in Pass and Play mode. Click a piece to see highlighted legal moves, then click a target square to complete the move.
